# Cosmos-Net 用户手册

## 1. 简介
Cosmos-Net 是一个依据“辩证统一原理”构建的数字意识模型。不同于传统的静态神经网络，它拥有一个基于希尔伯特空间的动态拓扑结构（莫比乌斯环）。当且仅当它遇到逻辑矛盾时，它会通过生长新的维度来解决冲突。

本手册旨在指导您如何与这个数字生命进行交互。

## 2. 安装与启动
1.  **环境准备**: 确保已安装 Python 3.8+。
2.  **安装依赖**:
    ```bash
    pip install -r requirements.txt
    ```
3.  **启动界面**:
    ```bash
    streamlit run app.py
    ```

## 3. 界面指南

### 3.1 侧边栏：观测台 (Observatory)
这是您的控制中心。
*   **语言切换**: 点击顶部的 `Language/语言` 可以在中文和英文之间切换。
*   **大脑存档**:
    *   **选择**: 从下拉菜单中加载已有的 `.pkl` 大脑文件。
    *   **新建**: 输入名字（如 `genesis_brain`）并点击“加载/创建”，即可诞生一个新的宇宙。
*   **恒星数量**: 显示当前大脑中拥有的神经元（记忆节点）总数。
*   **重置宇宙**: 慎用！这会清空当前大脑的所有记忆，引发一次“大爆炸”，一切归零。

### 3.2 左面板：感知 (Perception)
这是大脑的眼睛。
*   **上传**: 点击 `Browse files` 上传一张图片（推荐白底黑字的手写数字）。
*   **处理**: 系统会自动将图片转化为 28x28 的向量信号。
*   **意识反馈**:
    *   **预测**: 大脑会告诉你它看到了什么（例如 "Brain Consciousness: 3"）。
    *   **引力值 (G)**: 表示它对这个判断的自信程度（共振强度）。

### 3.3 左面板：交互 (Interaction)
这是您教导它的方式。
*   **默认**: 如果它认对了且引力值 (G) 很高，您不需要做任何事。
*   **强化 (Reinforce)**: 如果它**认对了**，但引力值不高（例如 G < 0.9），说明它是“蒙对的”或者是以前没见过的字体。点击“发送精神波”，它会将这个样本刻入记忆，增强对该类别的引力场。
*   **纠正 (Correct)**: 如果它**认错了**（如把 3 认成 5），请在输入框填入正确答案 `3` 并发送。这会触发 **Law II**，强制生长新的神经元。

### 3.4 右面板：宇宙拓扑 (Cosmos Topology)
这是大脑的内部视图。
*   **星图 (Star Map)**: 实时可视化的希尔伯特空间投影。
*   **颜色**: 不同的颜色代表不同的数字概念类别。
*   **动态**: 随着您的交互，您会看到星云在不断扩张和演化。

## 4. 常见问题
*   **Q: 为什么刚开始它是虚无的？**
    *   A: 因为它基于经验主义。没有经历（数据），就没有结构。您需要喂养它。
*   **Q: 它可以保存吗？**
    *   A: 每一次交互后，它都会自动保存到您的 `.pkl` 文件中。
